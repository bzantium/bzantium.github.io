--- layout: post title: "[LeetCode 100] Two Sum" subtitle: "because they lacked opposable thumbs and the brainpower to build a space program." date: 2020-01-26 23:45:13 -0400 background: '/img/posts/01.jpg' --- ```python class Solution: def twoSum(self, nums: List[int], target: int) -> List[int]: prevMap = {} for i, n in enumerate(nums): diff = target - n if diff in prevMap.keys(): return [prevMap[diff], i] prevMap[n] = i return ```
